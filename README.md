# online-food-order-analysis-in-India-in-the-months-of-January-February-and-March-2026
This is my first project on python where i have analyzed the data based on online food orders in India in months of January, February and March 2026. 
As my first project i think i have extracted enough information regarding the food ordering patterns based on the data.
I have analyzed various aspects on which food orders depends in the data.
i have made use of python libraries like numpy,pandas,seaborn and matplotlib.
i have made use of visualizations,groupby,sort,count and pivot_tables to extract as much data as possible.
i have made use of pie-charts,lineplot,countplot,pairplot,heatmaps and bar-charts to represent the visualizations.
